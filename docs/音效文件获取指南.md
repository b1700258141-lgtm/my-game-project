# 音效文件获取指南

本文档帮你逐一找到合适的 CC0 音效文件。所有文件放入 `public/assets/audio/sfx/`。

---

## 最快方案：Kenney Interface Sounds 包

**Kenney Interface Sounds** (CC0) 是覆盖 UI 按钮音效最快的包：
- https://kenney.nl/assets/interface-sounds
- 包含 click, hover, confirm, cancel, open, close 等
- 下载后解压，从中挑选合适的 WAV 重命名为对应文件名

**Kenney Casual Game Sounds** (CC0):
- https://kenney.nl/assets/casual-game-sounds
- 包含 coin, chime, pickup, success, fail 等

## 格式转换

如果下载的素材是 mp3/ogg，可用以下方式转为 WAV：

### 在线工具（免安装）
- https://convertio.co/mp3-wav/ (免费)
- https://online-audio-converter.com/

### Audacity（免费桌面软件）
- https://www.audacityteam.org/
- 打开文件 → 导出 → WAV
